Orbital

Play Demo!

Orbital is an Arcade Shooter / Tower Defence hybrid, in which you must defend your planet from incoming hordes of aliens.

Orbital was designed for one of my university courses, following a brief to make a demo for a free casual game. After spending some time creating a design document, the initial demo was created in 7 days and received a great response.

Development continues, and Orbital will be available to the public some time this year!

Key Features

  • Sprite animation
  • Enemy class hierarchy
  • Predictive aiming
  • Full object pooling
  • Detailed upgrade system
