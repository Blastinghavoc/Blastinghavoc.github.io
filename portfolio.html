Orbital

Play Demo!

Orbital is an Arcade Shooter / Tower Defence hybrid developed in Unity, in which you must defend your planet from incoming hordes of aliens.

Orbital was designed for one of my master's modules, following a brief to make a demo for a free casual game. After spending some time creating a design document, the initial demo was created in 7 days and received a great response.

Development continues, and Orbital will be available to the public some time this year!

Key Features

  • Sprite animation
  • Multiple enemy types
  • Predictive targetting
  • Full object pooling
  • Detailed upgrade system
  • Self-made sprites

Explorer

Explorer is a 2D platformer created in Monogame for one of my master's modules.

The focus of the module was Games Architecture, so Explorer is not just a game, but a small engine. The engine is designed to support any platformer game, with extensibility and modularity being major focuses of its design. The engine features collision detection and resolution, enemies, pickups, powerups and animations, all implemented from scratch in C# with the Monogame framework

Key Features

  • Event-driven architecture, maximising separation of concerns
  • Fully data-driven level and entity configuration via XML
  • Serialization for high-score tracking
  • Extensible enemy, pickup and projectile class hierarchies
  • Finite state machine based AI and animation control
  • Efficient asset management, loading and unloading assets between levels

3D_TD

3D_TD is a 3D Tower Defense game made using C++ and OpenGL for one of my master's modules.

A very small template was given that handled model loading and rendering, which I expanded into a working game demo. The player is able to design their own maze through which the enemies travel, and can place defensive turrets to help defeat them before they reach the exit portal.

Key Features

  • AI with pathfinding, flocking and predictive aiming behaviours
  • Grid-based maze and turret placement and removal
  • Imported meshes and animations, alongside OpenGL primitive-based objects
  • Combination of self-coded and Bullet physics

A.R.T (Advanced Racetrack Technologies)

A.R.T is another one of my master's projects using OpenGL and C++.

In A.R.T you drive a sci-fi vehicle around a winding 3D race track, collecting points and avoiding obstacles along the way. The main focus of the project was the graphics though, giving me experience with glsl shaders, multi-pass rendering techniques and procedural mesh generation.

Key Features

  • Procedurally generated track using a cubic spline
  • Instanced rendering
  • Post-processing techniques such as bloom, high dynamic range lighting and motion blur
  • GLSL programming for lighting and special effects
  • CPU-side particle effects

Damage Control

Play Demo!

Damage Control was created in Unity in 48 hours for Global Game Jam 2020 by a team of 5 City University students.

The theme of the Game Jam was "repair", so in Damage Control you play a wizard cleaning up after a hero as he crusades across the land. The gameplay consists of using your cursor to drag broken pieces of pots and crates together to repair them before they hit the ground. I was the main gameplay programmer implementing this, while other teammates focused on sound, graphics, UI and team management.

I enjoyed the intensity of the Game Jam experience and was very pleased with what the team achieved in the limited time.

Key Contributions

  • Click-and-drag object connection
  • Randomised prefab spawning
  • Trigger-based object fragmentation
  • Feature integration and merge conflict resolution for other teammates
